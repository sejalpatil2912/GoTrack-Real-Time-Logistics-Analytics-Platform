## 📖 Overview
GoTrack Enterprise is an end-to-end, industry-grade Logistics Management and Shipment Tracking Platform. Designed to solve challenges in the global supply chain, it features real-time shipment monitoring, role-based dashboards, automated risk prediction models, carbon footprint calculations, and a complete analytics export pipeline ready for Power BI visualization. 

## 🎯 Problem Statement
In modern supply chain management, simply tracking "Point A to Point B" is no longer sufficient. Enterprise logistics companies struggle with two major challenges:
1. **Unpredictable Delays**: Failing to account for weather conditions, priority levels, and transport modes resulting in unexpected delays.
2. **ESG Compliance (Sustainability)**: The inability to actively monitor and calculate the Carbon (CO2) footprint generated by global freight.

**GoTrack** solves this by providing a unified SaaS dashboard where clients can view their consignments in real-time, while Administrators utilize Data Science logic to automatically generate delay risk probabilities and calculate GHG carbon emissions dynamically.

## ✨ Core Features
- **Role-Based Access Control (RBAC):** Secure authentication separating Admin Operations from the secure Client Tracking Portals.
- **AI-Style Predictive Risk Scoring:** The system automatically analyzes Transport Mode (Air, Ocean, Road, Rail) and Weather Conditions (Storms, Snow, Clear skies) to generate a customized 0-100% Delay Probability score.
- **Automated Carbon Footprint Tracking:** Dynamic mathematical calculation of CO2 emissions generated, factored by freight weight and transit modality multipliers.
- **Power BI Analytics Pipeline:** A backend Pandas data processor that transforms backend MySQL data into refined `.csv` exports for instant Business Intelligence rendering.
- **"Enterprise Midnight" UI/UX:** Built on entirely customized Tailwind CSS blocks featuring glassmorphism elements, predictive visual progress bars, and glowing neon UI/UX aesthetics.

## 🏗️ System Architecture
The application runs on a Monolithic MVC (Model-View-Controller) architecture:
1. **Frontend (View):** Server-rendered Jinja2 HTML templates styled heavily with the Tailwind CSS Framework.
2. **Backend (Controller):** A Python Flask application managing API routing, Authentication (Werkzeug hashing), and Data Science logic generation.
3. **Analytics Engine:** Pandas data pipeline interacting with the database to clean and prepare data models for Power BI.
4. **Database (Model):** A relational MySQL database enforcing strict schemas using Foreign Keys to link Shipments strictly to registered Users.

## 🚀 How to Run the Program

### Prerequisites
- Python 3.10+ installed
- MySQL Server (e.g., XAMPP, WAMP, or MySQL Workbench) running locally on port 3306

### Installation Steps
   
1. **Activate the Virtual Environment**
   .\venv\Scripts\activate
     

2. **Install Dependencies**
   (If `requirements.txt` is present, run `pip install -r requirements.txt`. Otherwise install the core features:)
   
   pip install Flask PyMySQL pandas python-dotenv Werkzeug
   

3. **Run the Application**
  
   The application will boot up at `http://localhost:5000`.

### Data Injection (Optional)
